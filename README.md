# 博客更新流程
## 更新官网博客
1. 在项目blog目录下，新建xx-slug目录并粘贴完整内容到+page.md文件中，完成官网页面更新。
2. 更新官网static/sitemap.xml添加新页面。
3. 在搜索引擎（百度为主）手动添加索引。
